Houshang-Jani Kodi repository for add-ons to stream Indian OTT Content

Project maintained by Houshang-Jani



Add-ons

Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://Houshang-Jani.github.io/ (Give it the name "Houshang-Jani Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Houshang-Jani Repository", and install repository.Houshang-Jani.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Houshang-Jani Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
